# Cómo subir Plegar Pro a GitHub

## Crear el repositorio

1. En GitHub pulsa el botón **+** de la esquina superior.
2. Selecciona **New repository**.
3. Nombre recomendado: `plegar-pro`.
4. No marques la creación automática de README, `.gitignore` ni licencia.
5. Pulsa **Create repository**.

## Subir los archivos

1. Dentro del repositorio pulsa **Add file**.
2. Selecciona **Upload files**.
3. Arrastra todos los archivos y carpetas de este proyecto descomprimido.
4. Comprueba que `index.html` queda en la raíz del repositorio.
5. Escribe `Plegar Pro v19.8` como mensaje.
6. Pulsa **Commit changes**.

## Activar GitHub Pages

1. Abre **Settings**.
2. En el menú izquierdo abre **Pages**.
3. En **Source**, selecciona **GitHub Actions**.
4. Abre **Actions**.
5. Espera a que el despliegue tenga una marca verde.
6. En **Settings → Pages** aparecerá el enlace público.

## Importante

No subas solamente el archivo ZIP. GitHub Pages necesita los archivos descomprimidos y `index.html` en la raíz.
